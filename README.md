# 🎭 Telugu Dumb Charades

A simple web application that generates random Telugu movie titles for Dumb Charades.

Built for Telugu movie lovers who are tired of searching Google for movie names during game nights. Click a button and instantly get a random Telugu movie challenge from a database of nearly 2,000 Telugu movies.

## ✨ Features

- 🎬 Random Telugu movie generator
- 📚 1,958 Telugu movie titles
- 🔄 No-repeat movie generation
- 📱 Mobile-friendly design
- ⚡ Fast and lightweight
- 🌙 Clean dark-themed interface

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- JSON
- Wikipedia Movie Data

## 🚀 How It Works

1. Click **Generate Movie**
2. A random Telugu movie title appears
3. Act it out without speaking
4. Let your team guess the movie
5. Generate another movie and continue playing

## 💻 Run Locally

Clone the repository:

```bash
git clone https://github.com/Sharath8599/telugu-dumb-charades.git
```

Navigate into the project:

```bash
cd telugu-dumb-charades
```

Start a local server:

```bash
python3 -m http.server 8000
```

Open:

```text
http://localhost:8000
```

## 🎯 Why I Built This

My friends and I love playing Dumb Charades with Telugu movie names during parties and get-togethers.

Finding movie names quickly was always a hassle. This project solves that problem by providing a huge collection of Telugu movie titles that can be generated instantly with a single click.
